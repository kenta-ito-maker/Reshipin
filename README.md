# レピ (REPI) - AIおまかせごはん

冷蔵庫にあるものでAIが献立を提案するWebアプリ。

## 効果音について

効果音は外部ファイルを使用せず、**Web Audio API** でプログラム生成しています。

- タップ音：サイン波ピッチスイープ（880→1320Hz）
- 検索中（ジュウジュウ）：バンドパス＋ハイパスフィルタ付きノイズのループ再生、フェードイン/アウト
- 完成音：蒸気ノイズ（プシュー）+ ベル倍音（G6-D7-G7）+ ジングル（C-E-G-C↑）の3層

外部音源ファイルは一切使用していないため、ライセンスの制約はありません。
設定画面の「🔊 効果音」トグルでON/OFF切り替え可能です。

### iOS PWA対応

iOS Safari および PWA（ホーム画面に追加）モードでの音声再生に対応。
初回ユーザータッチ時にAudioContext初期化＋サイレントAudio要素のplay/pauseで
iOS の自動再生制限を回避しています。
